Instructions

This 2D puzzle game is meant to make you
think, but there’s not much of a point to thinking
if you don’t know the controls so to atleast play the game you
should know these basic controls. First off if you want to move you're going
to use the arrow keys. The ↑(Up) arrow key allows you to jump, To move wright you press
down on the →(wright) arrow key. Finally as you could probably
guess the left arrow key makes you go ← (left). Finally if you wanted to interact with a lever then all
you need to do is press X on your keyboard
